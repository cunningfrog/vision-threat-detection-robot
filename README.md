# Vision-Based Cyber-Physical Threat Detection Robot

🚀 AI-powered surveillance system for industrial infrastructure protection.

## 🔍 Overview
This project uses computer vision and AI to detect potential threats in real-time environments like power plants, data centers, and industrial control rooms.

## ⚙️ Features
- Real-time object detection using YOLOv8
- Anomaly detection system
- Threat scoring mechanism
- Secure alert system (encryption-based)

## 🧠 Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- NumPy

## 🏭 Use Cases
- Power Plants
- Data Centers
- Industrial Control Rooms
- Oil & Gas Facilities

## 📈 Future Improvements
- Integration with LLMs (Claude / GLM)
- Real-time dashboard (web interface)
- Voice-based alerts
- Cloud deployment
